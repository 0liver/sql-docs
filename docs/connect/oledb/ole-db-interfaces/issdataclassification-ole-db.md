# ISSDataClassification
[!INCLUDE [SQL Server](../../../includes/applies-to-version/sqlserver.md)]

[!INCLUDE[Driver_OLEDB_Download](../../../includes/driver_oledb_download.md)]

  **ISSDataClassification** interface provides the functionality to retrieve sensitivity classification data as received from SQL Server for the active rowset.
  
|Method|Description|  
|------------|-----------------|  
|[ISSDataClassification::GetSensitivityClassification](../../oledb/ole-db-interfaces/issdataclassification-getsensitivityclassification-ole-db.md)|Returns a pointer to an OLE DB Driver for SQL Server SENSITIVITYCLASSIFICATION structure that contains sensitivity classification information.|
