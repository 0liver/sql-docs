# Export Data from a DataFrame, insert via ODBC to SQL

This article will describe how to insert data into SQL Server from a Python Pandas DataFrame.  A connection is established to SQL Server using the open source module pyodbc. The data stored in the DataFrame can be inserted into a SQL Server database table.
