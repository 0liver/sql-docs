# sp_change_feed_enable_table (Transact-SQL)
[!INCLUDE [sqlserver2022-asdb](../../includes/applies-to-version/sqlserver2022-asdb.md)]

Adds a table and creates internal metadata objects for [Synapse Link for SQL Server](/azure/synapse-analytics/synapse-link/sql-server-2022-synapse-link). For more information, see [Manage Azure Synapse Link for SQL Server](../../sql-server/synapse-link/synapse-link-sql-server-change-feed-manage.md).

## Syntax  
   
 ![Topic link icon](../../database-engine/configure-windows/media/topic-link.gif "Topic link icon") [Transact-SQL Syntax Conventions](../../t-sql/language-elements/transact-sql-syntax-conventions-transact-sql.md)  
  
```syntaxsql  
EXECUTE sys.sp_change_feed_enable_table 
    @table_group_id uniqueidentifier,
    @table_id uniqueidentifier,
    @source_schema sysname,
    @source_name sysname
GO
```  

## Arguments  
- 
#### @table_group_id

Required. Data type is uniqueidentifier. 

The id of the change feed table group that the new link table will belong to. It is a GUID generated by the initial setup. Table_group_id must already exist in change feed. 

#### @table_id

Required. Data type is uniqueidentifier.

The id of the link table. It will be a GUID generated by the setup flow and passed to the source publisher during provision. 

#### @source_schema

Required. Data type is sysname.

The source table schema name. 

#### @source_name

Required. Data type is sysname.

The source table name. 

## Remarks

The source table must have a primary key.

## Permissions  

 Only a member of the sysadmin server role or db_owner database role can execute this procedure.
