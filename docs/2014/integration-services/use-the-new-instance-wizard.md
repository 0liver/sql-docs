# Use the New Instance Wizard
  The New Instance wizard is used to create a new instance for a CDC service. You open the Create an Oracle CDC Instance wizard from the CDC Designer Console. You can do the following in the New Instance wizard.  
  
-   [Create the SQL Server Change Database](../../2014/integration-services/create-the-sql-server-change-database.md)  
  
-   [Connect to an Oracle Source Database](../../2014/integration-services/connect-to-an-oracle-source-database.md)  
  
-   [Connect to Oracle](../../2014/integration-services/connect-to-oracle.md)  
  
-   [Select Oracle Tables and Columns](../../2014/integration-services/select-oracle-tables-and-columns.md)  
  
-   [Select Oracle Tables for Capturing Changes](../../2014/integration-services/select-oracle-tables-for-capturing-changes.md)  
  
-   [Make Changes to the Tables Selected for Capturing Changes](../../2014/integration-services/make-changes-to-the-tables-selected-for-capturing-changes.md)  
  
-   [Generate and Run the Supplemental Logging Script](../../2014/integration-services/generate-and-run-the-supplemental-logging-script.md)  
  
-   [Generate Mirror Tables and CDC Capture Instances](../../2014/integration-services/generate-mirror-tables-and-cdc-capture-instances.md)  
  
-   [Finish](../../2014/integration-services/finish.md)
