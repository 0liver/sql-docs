# Change Data Capture Designer for Oracle by Attunity
  The CDC Designer Console is used to develop and maintain Oracle CDC Instances. The CDC Designer Console is a Microsoft Management Console snap-in that contains the following elements:  
  
-   New Instance Wizard: This wizard creates a new Oracle CDC Instance. For information on using the New Instance Wizard, see [Use the New Instance Wizard](../../2014/integration-services/use-the-new-instance-wizard.md).  
  
-   CDC Instance Properties Viewer: This is a docked view showing the status and configuration of the selected CDC instance. For information about the properties viewer, see [How to Manage a CDC Instance](../../2014/integration-services/how-to-manage-a-cdc-instance.md).  
  
-   CDC Instance Properties Editor: This dialog box is used to edit any existing Oracle CDC Service instance. For information about editing the CDC instance properties, see [Edit Instance Properties](../../2014/integration-services/edit-instance-properties.md).
