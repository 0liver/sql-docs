# SQLDescribeCol (Visual FoxPro ODBC Driver)

> [!NOTE]  
> This article contains Visual FoxPro ODBC Driver-specific information. For general information about this function, see the appropriate article under [ODBC API Reference](../reference/syntax/odbc-api-reference.md).

## Support

Full.

## ODBC API conformance

Core level.

## Remarks

Returns the name, type, precision, scale, and nullability of the given result column.

For more information, see [SQLDescribeCol Function](../reference/syntax/sqldescribecol-function.md) in the *ODBC Programmer's Reference*.
