# SQLPutData (Visual FoxPro ODBC Driver)

> [!NOTE]  
> This article contains Visual FoxPro ODBC Driver-specific information. For general information about this function, see the appropriate article under [ODBC API Reference](../reference/syntax/odbc-api-reference.md).

## Support

Full.

## ODBC API conformance

Level 1.

## Remarks

Allows an application to send data for a parameter or column to the driver at statement execution time.

For more information, see [SQLPutData Function](../reference/syntax/sqlputdata-function.md) in the *ODBC Programmer's Reference*.
