# SQLCancel (Visual FoxPro ODBC Driver)

> [!NOTE]  
> This article contains Visual FoxPro ODBC Driver-specific information. For general information about this function, see the appropriate article under [ODBC API Reference](../reference/syntax/odbc-api-reference.md).

## Support

Full.

## ODBC API conformance

Core level.

## Remarks

Cancels the processing on a statement handle, `hstmt`.

For more information, see [SQLCancel Function](../reference/syntax/sqlcancel-function.md) in the *ODBC Programmer's Reference*.
