# SQLColumns (Visual FoxPro ODBC Driver)

> [!NOTE]  
> This article contains Visual FoxPro ODBC Driver-specific information. For general information about this function, see the appropriate article under [ODBC API Reference](../reference/syntax/odbc-api-reference.md).

## Support

Full.

## ODBC API conformance

Level 1.

## Remarks

For a table, creates a result set that is the column list for the specified table or tables.

For more information, see [SQLColumns Function](../reference/syntax/sqlcolumns-function.md) in the *ODBC Programmer's Reference*.
