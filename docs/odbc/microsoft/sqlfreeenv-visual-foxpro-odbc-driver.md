# SQLFreeEnv (Visual FoxPro ODBC Driver)

> [!NOTE]  
> This article contains Visual FoxPro ODBC Driver-specific information. For general information about this function, see the appropriate article under [ODBC API Reference](../reference/syntax/odbc-api-reference.md).

## Support

Full.

## ODBC API conformance

Core level.

## Remarks

Closes the Visual FoxPro ODBC Driver and releases all memory associated with the driver.

For more information, see [SQLFreeEnv Function](../reference/syntax/sqlfreeenv-function.md) in the *ODBC Programmer's Reference*.
