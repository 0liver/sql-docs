# SQLConnect (Visual FoxPro ODBC Driver)

> [!NOTE]  
> This article contains Visual FoxPro ODBC Driver-specific information. For general information about this function, see the appropriate article under [ODBC API Reference](../reference/syntax/odbc-api-reference.md).

## Support

Full.

## ODBC API conformance

Core level.

## Remarks

Connects to a data source, which can be either a [database](visual-foxpro-terminology.md) or a directory of [tables](visual-foxpro-terminology.md). The Visual FoxPro ODBC Driver ignores the `szUID`, `cbUID`, `szAuthStr`, and `cbAuthStr` arguments.

For more information, see [SQLConnect Function](../reference/syntax/sqlconnect-function.md) in the *ODBC Programmer's Reference*.
