# SQLExecute (Visual FoxPro ODBC Driver)

> [!NOTE]  
> This article contains Visual FoxPro ODBC Driver-specific information. For general information about this function, see the appropriate article under [ODBC API Reference](../reference/syntax/odbc-api-reference.md).

## Support

Full.

## ODBC API conformance

Core level.

## Remarks

Executes a prepared SQL statement (a statement already prepared by [SQLPrepare](sqlprepare-visual-foxpro-odbc-driver.md)). The driver uses the current values of the parameter marker variables if any parameters exist in the statement.

For more information, see [SQLExecute Function](../reference/syntax/sqlexecute-function.md) in the *ODBC Programmer's Reference*.
