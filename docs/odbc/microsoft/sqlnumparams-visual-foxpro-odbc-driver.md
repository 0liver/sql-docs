# SQLNumParams (Visual FoxPro ODBC Driver)

> [!NOTE]  
> This article contains Visual FoxPro ODBC Driver-specific information. For general information about this function, see the appropriate article under [ODBC API Reference](../reference/syntax/odbc-api-reference.md).

## Support

Full.

## ODBC API conformance

Level 2.

## Remarks

Returns the number of parameters in a SQL statement. The number of parameters should equal the number of question marks in the SQL statement passed to [SQLPrepare](sqlprepare-visual-foxpro-odbc-driver.md).

For more information about SQL grammar, see [Supported ODBC SQL Grammar](supported-odbc-sql-grammar-visual-foxpro-odbc-driver.md). For more information about this function, see [SQLNumParams Function](../reference/syntax/sqlnumparams-function.md) in the *ODBC Programmer's Reference*.
