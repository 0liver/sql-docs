# SQLNumResultCols (Visual FoxPro ODBC Driver)

> [!NOTE]  
> This article contains Visual FoxPro ODBC Driver-specific information. For general information about this function, see the appropriate article under [ODBC API Reference](../reference/syntax/odbc-api-reference.md).

## Support

Full.

## ODBC API conformance

Core level.

## Remarks

Returns the number of columns in a result set cursor.

For more information, see [SQLNumResultCols Function](../reference/syntax/sqlnumresultcols-function.md) in the *ODBC Programmer's Reference*.
